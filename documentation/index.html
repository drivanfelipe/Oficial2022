Structuring Elements

This theme is a Responsive HTML5 / CSS3 Template for Desktop, tablet, mobile phone…

Description:

- Sport Multipurpose HTML Template

- Rich UI Kit

- 2 Blog Layouts

- W3C valid, well commented and SEO friendly code

- E-Commerce Page Template

- Mega Footer

- And Much More...

 

This modern Pure Gym Responsive HTML5 Template will delight the visitors of your sport club for many years. It has an attractive slider showing people with athletic bodies. Four club programs are presented on cool square banners with ghost call-to-action buttons having rounded edges. Coaches profiles come with photos. Each customer testimonial is framed.

 

All of the information within the main content area is nested within a div with an class of "container". The general template structure is the same throughout the template. 

The html are commented well so you can easily make an update.
